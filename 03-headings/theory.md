# 3-dars — Headings (h1–h6)

## Headings nima?
Headings — bu sahifadagi sarlavhalar. HTML’da sarlavhalar `<h1>` dan `<h6>` gacha bo‘lgan teglar bilan yoziladi.

- `<h1>` — eng katta (asosiy) sarlavha
- `<h6>` — eng kichik sarlavha

Misol:

<h1>Bu h1 sarlavha</h1>
<h2>Bu h2 sarlavha</h2>
<h3>Bu h3 sarlavha</h3>
<h4>Bu h4 sarlavha</h4>
<h5>Bu h5 sarlavha</h5>
<h6>Bu h6 sarlavha</h6>

---

## Muhim qoida (strukturaga oid)
Odatda bitta sahifada **1 ta `<h1>`** bo‘ladi.
Keyin mavzular bo‘yicha tartib bilan pastga tushib boriladi:

<h1>Sayt nomi yoki sahifa sarlavhasi</h1>
<h2>Bo‘lim</h2>
<h3>Bo‘lim ichidagi qism</h3>
<h2>Keyingi bo‘lim</h2>

---

## Headings + paragraph
Sarlavhadan keyin matn odatda `<p>` tegi bilan yoziladi:

<h1>Frontend kursi</h1>
<p>Bu kursda siz HTML, CSS va JavaScript o‘rganasiz.</p>

<h2>HTML bo‘limi</h2>
<p>Bu bo‘limda HTML asoslari o‘rganiladi.</p>

---

## Ko‘p uchraydigan xatolar
1) `<h1>` ni ko‘p ishlatish (1 sahifada 3–4 ta h1)
2) Tartibni sakrab ketish (h1 dan keyin h4 qo‘yish)
3) Headings’ni faqat kattaroq font uchun ishlatish (dizayn uchun CSS kerak)
